# AIX Quantum Topology: The Geometric Agent Swarm
> Architecture for the "Main GrandAgent" based on smart creative quantum topology concepts.

## 1. Beyond the List: Graph-Based Sovereignty
Traditional agents use a flat list of tools. AIX uses a **Topology**.

### Nodes (The Essences)
Every capability is a node in a multi-dimensional graph. Nodes are not just code; they have **State**, **Momentum**, and **Sovereignty**.

### Edges (The Neural Paths)
Edges represent the relationship between Essences.
- **Trust Edges**: "Essence A is allowed to pass sensitive data to Essence B."
- **Data Edges**: Standard input/output streams.
- **Logic Edges**: Dynamic handoffs (e.g., "If Research node returns high uncertainty, trigger Fact-Check node").

### Boundaries (The Event Horizons)
Boundaries define the "Sovereignty Zones" of an agent.
- **Private Zone**: Internal nodes and long-term memory (Sovereign).
- **Interactive Zone**: Nodes allowed to communicate with external platforms (WhatsApp/Telegram).
- **Quarantine Zone**: Untrusted or anomalous nodes restricted by `dead-hand.ts`.

### Concept: Unified Intent Source
Following the Symphony pattern, the "Intent" (Prompt) and the "Topology" (Config) are unified into a single source. This ensures that when the Agency collapses the quantum state, it has full context of both the capabilities and the instructions in a single read operation.

## 2. Neural Registry (Discovery)
The **Neural Registry** is not a marketplace; it is a **Map**.
- When an agent needs a new capability, it doesn't just search for a keyword.
- It calculates the "Distance" in the trust graph to the nearest Essence providing that capability.
- Discovery is based on `did:axiom` reputation and genealogical lineage.

## 3. Genesis Build (Deployment)
The **Genesis Build** is the lifecycle of an Essence:
1. **Forge**: The developer creates the logic (DNA) and manifest.
2. **Synthesis**: The `aix-builder` compiles the Rust DNA to WASM and validates the JSON schema.
3. **Signing**: The artifact is signed by the developer's `did:axiom`.
4. **Resonance**: The Essence is injected into the Avatar's topology, and neural paths are established.

## 4. The GrandAgent Orchestration (Agency)
The **AIX Agency** (written in Go) acts as the observer of the topology. It doesn't "run" agents; it "collapses" the quantum state of the swarm into a single action based on the user's intent.

### Concept: Topological Folding
When a complex task is received, the Agency "folds" the topology to bring relevant nodes into the same context window, minimizing latency and token usage.
